問題解決

因為重啟的時候有看到如下的訊息

於是先使用這樣的做法

上面的作法等同於去修改 /etc/passwd 裡面 mysql 這個 user 的第六項資料，大致上是設定 mysql 這個 user 的家目錄的意思

再重啟訊息不見了，但是依然失敗
