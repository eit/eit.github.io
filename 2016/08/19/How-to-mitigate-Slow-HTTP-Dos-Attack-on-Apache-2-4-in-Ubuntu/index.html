How to mitigate Slow HTTP Dos Attack on Apache 2.4 in Ubuntu

緣起

拜請 Google 大神之後大致上看起來應該是不太難
主要參考了這兩篇：

  1. How do I disable directory browsing
  2. Apache 如何防護 Slow HTTP DOS 攻擊
